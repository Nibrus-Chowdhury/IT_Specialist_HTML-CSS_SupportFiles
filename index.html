Domain Summary

Domain 1 Summary

Domain 1 was a short introduction into the fundamentals of HTML. Some of the tags we learned were script, noscript, link, metadata, html, body, heading, and a little more. This domain taught us how to set up a basic html document along with linking it to CSS and adding script or noscript tags to alert the user if the website does or does not support javascript.

Domain 2 Summary

Domain 2 went more in-depth to the styling of a webpage through the stylesheet. We learned the correct syntax for CSS in order to implement style into a webpage, how to add IDs, how to add classes, and pseudo classes. This domain helps give us the necessary information needed to complete the next domains without confusion or repetition.

Domain 3 Summary

Domain 3 went very in-depth into the tags of HTML and overall what is capable with HTML. Some of the tags we learned were semantic tags, headers, paragraphs, divs, and much more. Along with this we learned how to make tables and how to input images along with hyperlinks. All these concepts tied in new ideas such as folder heirarchy in order to add links to images as well as niche concepts such as accessibilty to round out a good website or webpage.

Domain 4 Summary

Domain 4 went much more heavily into the media tags such as images, audio, track, and iframe as well as much more.
